---
title: "MCP"
description: "Connect ArcPay Somnia to local MCP hosts like Claude Desktop and Cursor."
---

ArcPay includes a local MCP server at:

```text
apps/mcp/server.mjs
```

Install dependencies:

```bash
npm run install:mcp
```

Run manually:

```bash
npm run mcp
```

Generate host config:

```bash
npm run arcpay -- mcp-config
```

## Tools

The MCP server exposes:

- `get_deployment`
- `derive_agent_id`
- `derive_invoice_id`
- `derive_claim_hash`
- `derive_privacy_commitment`
- `privacy_intent_guide`
- `invoice_guide`
- `x402_guide`
- `demo_path`
- `smoke_commands`

## Claude Desktop

Paste the output of `npm run arcpay -- mcp-config` into Claude Desktop's MCP server config, then restart Claude.

## ChatGPT

Standard ChatGPT web does not directly attach to local stdio MCP servers. For ChatGPT-style usage, use the public `/openapi.json`, `/llms.txt`, and HTTP tool wrapper.

## HTTP Tool Wrapper

List tools:

```bash
curl https://arcpay-somnia.vercel.app/api/developer/tools
```

Call a tool:

```bash
curl -X POST https://arcpay-somnia.vercel.app/api/developer/tools \
  -H "content-type: application/json" \
  -d '{"tool":"derive_agent_id","args":{"slug":"research-agent"}}'
```

URL form:

```bash
curl "https://arcpay-somnia.vercel.app/api/developer/tools/derive_agent_id?slug=research-agent"
```
